Olá, bem-vindo ao meu leia-me!

Como rodar meu projeto: 
****----1) Abra o arquivo .php (eu utilizei o servidor XAMPP);
****----2) Aguarde a mensagem "Abra o arquivo index.html!"; e
****----3) Quando aparecer, só abrir o index.html e a tabela estará lá!

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Do código HTML, tenho as seguintes observações a fazer:
Optei por usar a library do jQuery para fazer handling do código e a library  DataTable, dependente do jQuery, para handling da tabela.
Não tenho vivência suficiente com HTML para fazer minhas próprias funções para isso, então tive que ignorar a insturção de usar dados dentro de cada
<td> para mostrar a tabela. 
Optei por deixar meus scripts todos escritos no arquivo .html para evitar problemas com chamar o código de um arquivo .js.
*Sobre as sprites: acredito ser possível adicionar com o comando render, do DataTable, dados tanto da sprite como do nome associado a ela em outra coluna.
Não o fiz por ter medo de não dar tempo de enviar ainda hoje, na terça.

Do código em PHP, tenho as seguintes observações a fazer:
Não tive problemas para chamar os dados da API que, embora  bastante lenta às vezes, foi bem manejada pelo wrapper indicado no e-mail.
Porém, se fosse possível, iria preferir ter trabalhado diretamente em JavaScript para chamar e enviar dados orientados a objetos.
Não consegui chamar o arquivo .php pelo arquivo .html, utilizando um botão ou qualquer outro método. Não sei se foi por falta de experiência com rodar
servidores PHP integrados com HTML ou se estava realmente fazendo algum erro na chamada AJAX, então optei por rodar a página .php manualmente para poder
entregar dentro do prazo.

Foi uma experiência bem bacana, no geral, pois nunca havia feito um projeto integrando três linguagens que, por mais integrada que suas interações sejam,
tem diversas peculiaridades nas implementações e na relação uma com as outras. Me ajudou a perceber o escopo em que se pode trabalhar com programação e na
implementação de desenvolvimento.

Dei o meu melhor priorizando conseguir apresentar o código funcionando, espero ter conseguido mostrar suficientemente minhas habilidades!

Até a próxima!
